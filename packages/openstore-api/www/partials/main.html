Welcome to the OpenStore for Ubuntu Touch!

It’s time to get some hacking started! This is an attempt to bootstrap some hacking community around the devices that wants to explore the phone beyond of what the platform offers to the normal user at this time. The OpenStore allows hackers to install applications that can’t be published to the official store. This really is not about competing with the official Ubuntu store, but is meant as an extension for power users and developers.

The OpenStore opens doors for possibilities to break your Ubuntu installation. Only install it if you know what you’re doing, install it at your own risk!

Join us on freenode #ubuntu-openstore or our mailing list.
